# 📊 Advanced Company Data Analysis

A Python-based data analysis project that uses **NumPy** and **Pandas** to analyze a company dataset containing **600+ employee records**. The project performs data cleaning, statistical analysis, employee performance evaluation, and generates business insights through automated reports.

---

## 🚀 Features

* Load and clean employee dataset
* Remove duplicate records
* Detect missing values
* Calculate Profit and Revenue per Hour
* Generate Productivity & Efficiency Scores
* Classify employees by Performance Level
* Identify Business Risk
* Department-wise Revenue, Profit & Rating Analysis
* Top & Bottom Employee Analysis
* Employee Revenue Ranking
* Revenue Percentile & Quartile Analysis
* Correlation Matrix
* Pivot Table Analysis
* Statistical Analysis using NumPy
* Export final analysis report to CSV

---

* Python 3
* Pandas
* NumPy

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Advanced-Company-Data-Analysis.git
```

2. Navigate to the project folder

```bash
cd Advanced-Company-Data-Analysis
```

3. Install the required libraries

```bash
pip install pandas numpy
```

4. Run the project

```bash
python analysis.py
```
